Today's study summary:

**JQuery:**
#### JQuery event methods: 
    UI: focus, blur
    Keyboard: input, keydown, keyup, keypress
    Mouse: click, mouseover
    Form: submit

#### JQuery event object:
    event.type
    event.which
    event.data
    event.target

#### Basic Effect:
    .show() / .hide() /.toggle()
    .fadeIn() / .fadeOut() / .slideUp() / .slideDown()

#### Animation:
    $(element).animate([effect],[,speed],[,easing],[,complete]);
